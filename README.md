# 🌿 Greenden - Responsive Plant Website

Greenden is a responsive multi-page website built using **HTML** and **Tailwind CSS**. The project features a clean, modern user interface and demonstrates how to create responsive layouts using Tailwind's utility-first approach. It is designed as a learning project to practice building real-world websites with responsive design principles.

---

## 📖 Overview

Greenden introduces users to a beautifully designed plant-themed website with a responsive layout that works seamlessly across different screen sizes. The project focuses on creating clean, maintainable code while leveraging Tailwind CSS for styling.

---

## ✨ Features

- 🌿 Responsive multi-page website

- 📱 Mobile-first responsive design
- ⚡ Built entirely with HTML and Tailwind CSS
